By including Enumerable in a class, you get more power to your class!

  • methods packed into the Enumerable module are mixed into the methods inherited with the class Array / Hash / Range.
  • the array is the most generic way to spit out the results from enum methods like select or filter.

What does #map do?

map will return a new array based on the code block that is executed on each element of the collection. A few key points about the map method in Enumerable module:

  • always returns an array with the same quantity of elements as the original array
  • each element has been run through the code block
  • return is KING with #map

see #map run

array = [2,4,6]
result = array.map { |num| num**2 }

Notice that the code block ran on each element and that our result is an array with the returned values. The code was executed and now our array displays that result.

When would you use #map? When you want access to the return value of your code block.note: this will only make sense if you have a strong understanding of the difference between puts, print and return.
