  • What are the benefits of version control?
    • Wait, what is version control? Marley, Diddy and Beyonce are a team and working on a project together and have one file: Change the World through Dance. Version control allows each contributor to work on the project without interrupting someone else's work and maintain the integrity of each person's unique work. It eliminates the need to send the newest version through email or save another copy to Dropbox or another cloud storage place. Let's imagine B, Diddy and Marley all adding text and inserting sounds at the same moment in time. But oops! Baby Marley hit all the keys on his dad's keyboard and did some funky stuff. No worries! Version control maintains each saved point because life happens.
  • How does git help you keep track of changes?
    • Git is the software on your computer that allows you to save and keep track of your edits as you go. Let's say you are writing a blog entry (ha!) and have lots of funny ideas so you dump them into a file. You save just the initial ideas. The next day you work on developing a story around Funny Idea about Kale. Save with git. That means git has a version of your file before Kale and after Kale.
