Classes in Ruby

27 de mayo 2015

So, you've been working with strings and variables, the objects, and applying methods to them. Every object is part of a class. FixNum is the class for numbers, Array is the class about arrays. That means when I set up an array, I am inheriting the characteristics established in the class Array. Let's go one step further: if Ruby doesn't have a class for something I need, I can create one.

Classes are powerful! Think of a class as a parent: the object that provides the DNA. Maybe you inherited your green eyes from your dad and your long legs from your Mom. But your behavior: how you act and react in this crazy world, that is all you. Same with the ojects in Ruby. The class provides the structure but when you instantiate an object from a class, you set the unique behaviors of that class.

Some important things I've learned about classes:

  • a class holds instance variables, unique to that class. If succulents are a type of class, all the different types of succulents are instances - or unique objects - of that category. Go a step up: you've got a class of plants.
  • a class is initialized when you call it from outside. So, a class doesn't just produce its own object by being there. You want a new Plant? Create one! How?new_plant = Plant.new("sunflower").
  • a class can talk to other classes.
  • inside the class are the instance variables and methods
  • each instance of a class is unique and its own object.

What's a Class?

Let's talk about Plants again and start simple.

class Plants def initialize(plant) @plant = plant end end #end of class new_plant = Plants.new("sunflower")

What just happened? new_plant is our newly established instance of plant. A sunflower plant exists.

Now let's do something with our sunflower plant. class Plants def initialize(plants) @plant = plant end def height height = rand(1..10) puts "Your #{@plant} is #{height} centimeters tall." end def bloom @petals = rand(3..20) if @petals > 10 puts "It is spring! Your #{@plant} has #{@petals} petals." else puts "Must be the end of summer, you only have #{@petals} petals." end end
Now we can create an instance of the class Plants AND do something!
plant_1 = Plants.new("daffodil") plant_1.height => will give us the height. This is what I got in irb:irb(main):012:0> plant_2 = Plants.new("tulip") => # irb(main):013:0> plant_2.height Your plant is 27 cm tall.
